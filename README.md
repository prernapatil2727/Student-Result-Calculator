
# 🎓 Student Result Calculator

## 📖 Project Overview
Student Result Calculator is a simple web application developed using HTML, CSS, and JavaScript. It allows users to enter marks for 5 subjects and calculates the total marks, percentage, grade, and pass/fail status.

## ✨ Features
- ✅ Input marks for 5 subjects
- ✅ Validation for marks between 0 and 100
- ✅ Calculate total marks
- ✅ Calculate percentage
- ✅ Display grade (A, B, C, D, F)
- ✅ Display pass/fail status
- ✅ Responsive and user-friendly interface

## 🛠️ Technologies Used
- 🌐 HTML
- 🎨 CSS
- ⚡ JavaScript

## 🚀 How to Run
1. Download or clone the repository.
2. Open the `index.html` file in any web browser.
3. Enter marks for all 5 subjects.
4. Click the **Calculate Result** button.
5. View the results instantly.

## 📊 Grade Criteria

| Percentage | Grade |
|------------|--------|
| 90% and above | A |
| 75% - 89% | B |
| 60% - 74% | C |
| 40% - 59% | D |
| Below 40% | F |

## 🎯 Pass Criteria
A student is considered **Pass** if they score at least **35 marks in each subject**; otherwise, the result is **Fail**.
